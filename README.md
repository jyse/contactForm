# Contact Form: React + PHP

**start React**
```
npm start
```

**start PHP**
```
php -S localhost:8000
```

**Test with Browser**
```
visit http://localhost:3000
```

**Test with Postman**
```
GET http://localhost:8000


OPTION http://localhost:8000


POST http://localhost:8000
Content-Type: application/json
{
	"fname":"Hello",
	"lname":"World",
	"email":"hello@world.com",
	"message":"abc",
	"mailSent":false,
	"error":""
}
```
